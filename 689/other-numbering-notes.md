# Understanding the Rentcammer Filing System in Schaumburg-Lippe

```{note}
**#10** in the Acten Designation is **not a memo**; it’s the **Obligation (the sealed
promissory note itself)**. Originals of **promissory notes** were often
**kept in a separate “obligations” series** and, after repayment,
**returned to the debtor**—so they commonly **don’t carry** the file’s boxed
corner number and may not be physically in the case file anymore.
*That’s why #10 has no boxed number in your file.*
```

## Why there are multiple numbers/stamps on the same sheets

Different desks used **different books** to track the same matter. Each book created its **own** number/mark. They
use **parallel systems.** An **NC** number is **not** the same thing as **Nr pr** or the **boxed corner number**.

**Key takeaways about numbering systems**:

- **They are parallel systems.** An **NC** number is **not** the same thing as **Nr pr** or the **boxed corner number**.
- It’s normal that **only one** ledger/protocol number (e.g., **Nr pr: 1274**) shows on a single leaf.
- **“Nro g. d.”** is frequently just a tight **rubric** on copies/minutes; the number/date are recorded in the registry
book and/or noted elsewhere.

## Other small marks you’ll see (quick glossary)

- **Conc. + initials** (e.g., **Conc. G. W. Fuchs.**) — **Concordat**: “collated; agrees with the original,” attested by the clerk; written on the **office copy**.
- **(L.S.)** — **locus sigilli** (“place of the seal”).
- **F. p.** = Für protokoll - This document has been officially recorded and put on file
